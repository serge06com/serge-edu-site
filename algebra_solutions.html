Grade 12 Algebra – Equations and Inequalities (Solutions)

  1. 1. Solve for x: x² - 5x + 6 = 0 
    Factor the quadratic:
(x - 2)(x - 3) = 0

Set each factor equal to zero:
x - 2 = 0  →  x = 2
x - 3 = 0  →  x = 3

Solutions: x = 2 or x = 3
  2. 2. Solve for x: x² + 3x - 10 = 0 
    Factor the quadratic:
(x + 5)(x - 2) = 0

Set each factor equal to zero:
x + 5 = 0  →  x = -5
x - 2 = 0  →  x = 2

Solutions: x = -5 or x = 2
  3. 3. Solve for x: 2x² - 7x + 3 = 0 
    Use quadratic formula: x = [-b ± √(b² - 4ac)] / 2a
Here, a=2, b=-7, c=3

Discriminant:
D = (-7)² - 4(2)(3) = 49 - 24 = 25

Solutions:
x = [7 ± √25] / (2 * 2)
x = (7 ± 5) / 4

x₁ = (7 + 5) / 4 = 3
x₂ = (7 - 5) / 4 = 0.5

Solutions: x = 3 or x = 0.5
  4. 4. Solve for x: (x - 4)(x + 2) = 0 
    Set each factor equal to zero:
x - 4 = 0  →  x = 4
x + 2 = 0  →  x = -2

Solutions: x = 4 or x = -2
  5. 5. Solve for x: x(x - 3) = 10 
    Expand:
x² - 3x = 10

Bring all terms to one side:
x² - 3x - 10 = 0

Factor:
(x - 5)(x + 2) = 0

Solutions:
x = 5 or x = -2
  6. 6. Solve for x: x² - 2x = 8 
    Bring all terms to one side:
x² - 2x - 8 = 0

Factor:
(x - 4)(x + 2) = 0

Solutions:
x = 4 or x = -2
  7. 7. Solve the inequality: x² - 9 < 0 
    Rewrite:
(x - 3)(x + 3) < 0

Critical points: x = 3 and x = -3

Test intervals:
- For x between -3 and 3, inequality holds.

Solution:
-3 < x < 3
  8. 8. Solve the inequality: (x - 3)(x + 5) > 0 
    Critical points: x = 3 and x = -5

Test intervals:
- For x < -5, both factors negative → product > 0
- For -5 < x < 3, factors opposite signs → product < 0
- For x > 3, both factors positive → product > 0

Solution:
x < -5 or x > 3
  9. 9. Solve for x: x² - 7x + 10 = 0 
    Factor:
(x - 5)(x - 2) = 0

Solutions:
x = 5 or x = 2
  10. 10. Solve for x: 3x² + 2x - 1 = 0 
    Use quadratic formula: a=3, b=2, c=-1

D = 2² - 4(3)(-1) = 4 + 12 = 16

x = [-2 ± 4] / (2*3)

x₁ = ( -2 + 4 ) / 6 = 1/3
x₂ = ( -2 - 4 ) / 6 = -1

Solutions: x = 1/3 or x = -1

Grade 12 Algebra – Quadratic Equations (Solutions)

  1. Solve for x: x² - 5x + 6 = 0
    Solution: Factor: (x - 2)(x - 3) = 0
    So, x = 2 or x = 3
  2. Find the roots of 2x² + 3x - 2 = 0
    Solution: Use quadratic formula: a=2, b=3, c=-2
    x = [-3 ± √(9 + 16)] / 4 = [-3 ± 5]/4
    Roots: x = 1/2 or x = -2
  3. Solve: x² + 4x + 4 = 0
    Solution: Factor: (x + 2)² = 0
    So, x = -2

Grade 12 Quadratic Number Patterns (Solutions 21–30)

21. Find the 12th term of the sequence: 3, 8, 15, 24, ...
First differences: 5, 7, 9, ... (increasing by 2) → second difference = 2 (constant)
General form: \( a_n = An^2 + Bn + C \)
Using terms:
\( a_1 = A(1)^2 + B(1) + C = 3 \)
\( a_2 = 4A + 2B + C = 8 \)
\( a_3 = 9A + 3B + C = 15 \)
Solving system:
From second difference = 2 → \( 2A = 2 \) → \( A = 1 \)
Substitute \( A = 1 \) into equations:
\( 1 + B + C = 3 \Rightarrow B + C = 2 \)
\( 4 + 2B + C = 8 \Rightarrow 2B + C = 4 \)
Subtract first from second: \( (2B + C) - (B + C) = 4 - 2 \Rightarrow B = 2 \)
Then \( C = 2 - B = 0 \)
Formula: \( a_n = n^2 + 2n \)
Find \( a_{12} = 12^2 + 2 \times 12 = 144 + 24 = 168 \)
Final answer: 168

24. Find the sum of the first 7 terms of: 1, 4, 9, 16, ...

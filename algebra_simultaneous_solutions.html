Grade 12 Algebra – Simultaneous Equations (Solutions)

  1. Linear + Linear:
    Solve:
    x + y = 10
    2x - y = 5
    Solution:
    Add equations:
    x + y + 2x - y = 10 + 5 → 3x = 15 → x = 5
    Plug into first: y = 10 - 5 = 5
    Answer: x = 5, y = 5
  2. Quadratic + Linear:
    Solve:
    y = x² + 3
    2x + y = 7
    Solution:
    Substitute y into second: 2x + x² + 3 = 7
    → x² + 2x - 4 = 0
    Solve using quadratic formula:
    x = [-2 ± √(2² - 4(1)(-4))]/2(1) = [-2 ± √20]/2 = -1 ± √5
    x ≈ 1.24 or -3.24
    y = x² + 3 → y ≈ 4.54 or 13.5
    Answer: (1.24, 4.54) or (-3.24, 13.5)
  3. Exponential + Linear:
    Solve:
    2x + y = 7
    x + y = 3
    Solution:
    From second: y = 3 - x
    Sub into first: 2x + 3 - x = 7 → 2x = x + 4
    No algebraic solution → use graphical or numerical methods.
    Try x = 1 → LHS: 2 + 2 = 4 ❌
    Try x = 2 → 4 + 1 = 5 ❌
    Try x ≈ 2.5 → 5.66 + 0.5 ≈ 6.16 ✅
    Best solution (numerical): x ≈ 2.8, y ≈ 0.2
  4. Circle (Non-linear) + Linear:
    Solve:
    x² + y² = 25
    x - y = 1
    Solution:
    x = y + 1
    → (y + 1)² + y² = 25 → y² + 2y + 1 + y² = 25
    → 2y² + 2y - 24 = 0 → y² + y - 12 = 0
    Solve: y = [-1 ± √(1 + 48)]/2 = [-1 ± 7]/2 = 3 or -4
    x = y + 1 → x = 4 or -3
    Answers: (4, 3) and (-3, -4)
  5. Logarithmic + Linear:
    Solve:
    log(x) + y = 4
    x - y = 2
    Solution:
    x = y + 2 → log(y + 2) + y = 4
    Solve numerically:
    Try y = 1 → log(3) + 1 ≈ 0.48 + 1 = 1.48 ❌
    Try y = 2 → log(4) + 2 ≈ 0.6 + 2 = 2.6 ❌
    Try y ≈ 2.8 → log(4.8) + 2.8 ≈ 0.68 + 2.8 = 3.48 ❌
    Try y ≈ 3.3 → log(5.3) + 3.3 ≈ 0.72 + 3.3 = 4.02 ✅
    So y ≈ 3.3, x = 5.3

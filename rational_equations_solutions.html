📙 Grade 12 Algebra – Rational Expressions & Equations (Solutions)

  1. Simplify: (x² - 9)/(x + 3)
    → Factor numerator: (x - 3)(x + 3)/(x + 3)
    → Cancel common term: x - 3
  2. Find the excluded values: (x + 2)/(x² - 4)
    → x² - 4 = (x - 2)(x + 2); Denominator ≠ 0
    → Excluded values: x ≠ 2 and x ≠ -2
  3. Solve: 1/(x + 1) = 2/(x + 3)
    → Cross-multiply: (x + 3) = 2(x + 1)
    → x + 3 = 2x + 2 → x = 1
  4. Simplify: (x² - 16)/(x² - 4x)
    → (x - 4)(x + 4) / x(x - 4)
    → Cancel x - 4: Answer = (x + 4)/x
  5. Solve: (x + 2)/(x - 3) = (2x + 1)/(x² - 3x)
    → x² - 3x = x(x - 3)
    → Rewrite both sides: (x + 2)/(x - 3) = (2x + 1)/x(x - 3)
    → Multiply both sides by x(x - 3):
    → x(x + 2) = 2x + 1 → x² + 2x = 2x + 1 → x² = 1 → x = ±1
